httpie-http2
============

Experimental HTTP/2 plugin for `HTTPie <http://httpie.org>`_.


Installation
------------

.. code-block:: bash

    $ pip install -U httpie httpie-http2


Usage
-----

.. code-block:: bash

    $ http https://nghttp2.org/httpbin/get


.. image:: screenshot.png
    :alt: HTTPie HTTP/2
    :align: center


Requirements
------------

* Python 2.7.9 or 3.4+
* hyper_

.. _hyper: https://github.com/Lukasa/hyper
